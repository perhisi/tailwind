# Portfolio Website - Perhisi Ardhian

A personal portfolio website showcasing professional experience, projects, and skills as a Full-Stack Software Engineer.

## 🚀 Features

- **Responsive Design** - Works on all devices (mobile, tablet, desktop)
- **Dark/Light Mode** - Automatic theme detection based on system preferences
- **Smooth Scrolling** - Navigation with smooth scroll behavior
- **Interactive Elements** - Hover effects, animations, and transitions
- **Project Showcase** - Detailed project pages with tech stack and descriptions
- **Experience Timeline** - Paginated experience section with arrow navigation and scrollable content

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework (v4 via CDN)
- **JavaScript** - Interactive functionality
- **Google Fonts** - JetBrains Mono font family

## 🎨 Sections

1. **Header** - Logo, navigation menu, social links
2. **Hero** - Introduction and call-to-action
3. **About** - Personal background and skills
4. **Education** - Academic qualifications
5. **Experience** - Work history with pagination
6. **Projects** - Portfolio showcase
7. **Contact** - Contact form and social links
8. **Footer** - Copyright and quick links
